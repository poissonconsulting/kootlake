# Bull Trout Counts

The annual resistivity counter and redd counts for Kaslo and Keen
Creeks.

## Usage

``` r
bulltrout
```

## Format

A tbl data frame:

- Year:

  The year (int).

- KasloRedds:

  The Kaslo Creek redd count (int).

- KeenRedds:

  The Keen Creek redd count (int).

- KasloCounter:

  The Kaslo Creek resistivity counter count (int).
